### Project Description
This Daily Planner website helps users organize their day by managing priorities, schedules, focus sessions, and revision tasks in one place. It includes live date/time tracking and reminder notifications to improve productivity. The colorful interface makes planning simple and engaging.

### The Problem statement
Many students struggle with managing their daily tasks, organizing schedules, and remembering important activities. This often leads to poor time management, missed tasks, and reduced productivity.
### The Solution
The Daily Planner website provides a simple, structured platform to plan priorities, organize time blocks, track focus and revision tasks, and set reminders. This helps users stay organized, manage time effectively, and improve productivity.
